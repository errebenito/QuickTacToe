QuickTacToe
===========
Tic Tac Toe game written in C++ using Qt (QML/QtQuick).

Build instructions
==================
1. Download source code
2. mkdir build
3. cd build
4. qmake ..
5. make

After that, just type *./QuickTacToe* to run the game.
